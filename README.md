# Deja vu for Vue 2

> A very light Vue based UI library

## install
```bash
npm install deja-vu --save-dev
```

## Quick Start
```javascript
import Vue from 'vue'
import DejaVu from 'deja-vu'

Vue.use(DejaVu)
```
